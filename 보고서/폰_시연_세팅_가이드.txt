  폰 시연 세팅 가이드
  "휴대폰으로 PC의 클로드를 조종하는 시연 준비"
══════════════════════════════════════════════════════════════


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 사전 준비물
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ PC: Claude Code (CLI) 설치 완료 (현재 사용 중)
  ✅ 폰: Claude 앱 설치 (iOS App Store / Google Play)
  ✅ 동일한 Anthropic 계정으로 PC·폰 모두 로그인
  ✅ PC와 폰 모두 인터넷 연결 상태


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 방법 1. Claude 원격 세션 (Remote Session)
 → 폰에서 PC의 파일을 직접 조작하는 시연용
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [PC에서 할 일 - 교육 시작 전]

  1단계: 터미널(VS Code 터미널)에서 실행
    claude --remote

  2단계: 화면에 나오는 세션 URL 복사
    예시: https://claude.ai/remote/abc123...

  3단계: 이 URL을 폰으로 전송
    → 카카오톡 나에게 보내기 / 메모 앱 등

  [폰에서 할 일]

  4단계: 폰 브라우저에서 세션 URL 접속
    → PC의 Claude Code와 연결됨

  5단계: 폰에서 입력:
    "보고서 폴더에 있는 파일들 전부 읽고 요약해줘"

  6단계: PC 화면에서 클로드가 파일을 읽는 모습이 실시간 표시!
    → 교육생들에게 PC 화면을 빔프로젝터로 보여주면 됨


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 방법 2. Claude 프로젝트 활용 (간편 버전)
 → 원격 세션이 안 될 경우 대안
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [사전 세팅 - 1회만]

  1. claude.ai → 프로젝트 → 새 프로젝트 만들기
  2. 프로젝트 이름: "매일 아침 뉴스 브리핑"
  3. 프로젝트 지시사항에 아래 내용 붙여넣기:
     (클로드_프로젝트_지시사항.txt 파일 참고)
  4. 보고서 폴더의 파일들을 프로젝트 지식에 업로드

  [폰에서 시연]

  1. Claude 앱 → 해당 프로젝트 선택
  2. "브리핑" 입력 → 뉴스 요약 수신
  3. "보고서 요약해줘" → 업로드된 파일 기반 요약

  ※ 차이점: 방법1은 PC 파일을 실시간 접근,
            방법2는 미리 업로드한 파일만 접근 가능


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 시연 당일 체크리스트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  교육 30분 전:
  □ PC 전원 ON, 절전모드 해제
  □ VS Code 또는 터미널 열기
  □ claude --remote 실행
  □ 세션 URL을 폰으로 전송
  □ 폰에서 URL 접속 확인
  □ 테스트 메시지 전송해서 연결 확인
  □ PC 화면을 빔프로젝터에 연결

  시연 직전:
  □ 폰 화면: 교육생들에게 보여줄 수 있게 준비
  □ PC 화면: 빔프로젝터로 교육생들에게 표시
  □ "보고서" 폴더에 시연용 파일 3개 확인

 시연 시 멘트 (대본)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  "자, 지금 제 손에 있는 건 그냥 휴대폰입니다.
   그리고 저기 화면에 보이는 건 제 PC입니다.

   제가 폰에서 이렇게 치겠습니다.
   '보고서 폴더에 있는 파일들 전부 읽고 요약해줘'

   (폰에서 입력)

   자, PC 화면 보시죠.
   클로드가 지금 제 PC에 있는 파일을 하나씩 읽고 있습니다.
   저는 지금 이 자리에서 폰만 들고 있는데,
   제 책상 위 PC가 알아서 일하고 있는 겁니다.

   출근길 지하철에서 이렇게 하면,
   회사 도착했을 때 이미 요약이 끝나 있겠죠?"
